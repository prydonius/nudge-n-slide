NudgeNSlide v2.0


Description

Nudge 'n Slide is a customisable, and easy to use slider jQuery plugin. It can be used to slide through anything, videos, text, images. It aims to be incredibly easy to customise for advanced web developers, as well as easy to setup for beginners.

Click next for features

Features: New in 2.0!

  • You can now display slides of different sizes
  • Control the slider: play, pause, nudge when you want
  • Nudge to any slide you want (not just next/previous)
  • The 'important' css is done for you, all you have to do is style the slider
  • You don't need to assign a class to all of your slides anymore
